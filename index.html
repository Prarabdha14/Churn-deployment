Live Customer Churn Predictor

An interactive ML application demonstrating a real-time prediction and retraining pipeline.

Predict Churn for a New Customer

How It Works

  1. 1. The model is trained live on a sample dataset when the app starts.
  2. 2. Submit a new customer's data to get an instant churn prediction.
  3. 3. Add a full customer record to the database, which automatically retrains the model with the new data.

Add Customer & Retrain Model

Use the main form on the left. Once you have a prediction, add a known "Churn" status (Yes/No) and submit it here.
